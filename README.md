# Windows Server Administration Lab

Hands-on lab demonstrating core Windows Server administration tasks commonly performed by system administrators and IT support engineers.

## Lab Environment

Windows Server 2019 / 2022  
Active Directory Domain Services  
DNS  
Group Policy  
PowerShell  

## Tasks Performed

### Active Directory Setup
- Installed Active Directory Domain Services
- Promoted Windows Server to Domain Controller
- Created Organizational Units
- Created and managed domain users

### Group Policy Management
- Created and applied Group Policy Objects
- Enforced password policies
- Configured login restrictions
- Applied desktop policies

### DNS Configuration
- Configured DNS zones
- Verified name resolution
- Troubleshooted DNS issues

### User and Group Management
- Created security groups
- Assigned user permissions
- Managed domain access rights

### PowerShell Administration
- Automated user management
- Executed administrative tasks using PowerShell scripts
- Performed system checks using PowerShell commands
